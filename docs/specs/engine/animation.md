# Animation System

> Status: draft
> Crate: amigo_animation
> Depends on: [engine/core](../engine/core.md)
> Last updated: 2026-03-16

## Zweck

Sprite-based animation system for the Amigo Engine, driving frame-based animations from Aseprite data.

## Verhalten

Sprite animations from Aseprite tags. Frame-based with per-frame duration (fixed-point). Looping/one-shot modes. Phase 2: skeletal animation for large bosses.
